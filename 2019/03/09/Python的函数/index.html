五、局部变量 全部变量

  • 局部变量：只在函数内有用，出了这个函数就没用了
  • 全部变量：在函数外 整个文件内都可以访问全局变量

  • 不要轻易在函数内修改全部变量，因为就算修改全局变量，python会使用屏蔽（shadowing）方法来保护全局变量，新建一个同名的局部变量，原本的全局变量不变。

  • global：如果一定想好要修改全局变量，采用global形式

六、内嵌函数和闭包

1.内嵌函数：在函数内部创建另一个函数

2.闭包：函数式编程语言

对于funy来说，funx就是它的外部函数，funy等就是funx的内部函数，且在内部对于x有引用，funy称为闭包函数。
